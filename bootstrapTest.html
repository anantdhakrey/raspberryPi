t#5 utility classes

margin and padding: goes from 1 to 5. my for margin in y direction, mx for x dir. mx-5. for each dir separately use: top- mt, btm- mb, left-ms for start and right-me for end. similarly for padding

to get the border, just write the border class. for individual border, use border-top, border-end, etc. border-success for border color. border-5 for thickness (1 to 5) . use rounded for rounded edges. rounded-pill for pill shape. shadow-sm for small shadow in border element. use lg for large shadow and only shadow for medium shadow

container options: container (max-width based on breakpoints), container-fluid (always 100% width), contianer-lg (100% width until lg and then max width based on breakpoint. instead of lg we can use sm,md,xs,xl, etc

grid goes inside a container. then row and then cols.
